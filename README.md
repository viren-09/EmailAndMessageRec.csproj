# .NET Email Sender API

A simple .NET application to send emails using a third-party email service provider (e.g., SendGrid, Mailgun, or SMTP-based services).  
This project demonstrates how to integrate external APIs securely with .NET.

---

##  Features
- Send emails programmatically using .NET
- Secure API key integration
- Configurable sender and receiver addresses
- Error handling and logging
- Easily extendable for attachments, HTML content, and templates

---

##  Prerequisites
- .NET 6.0 or later
- A third-party email service account (e.g., SendGrid, Mailgun, SMTP provider)
- API key from your chosen provider
